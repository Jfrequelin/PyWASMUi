# Documentation pyWasm (FR)

Cette section contient la documentation complete en francais du projet `pyWasm`.

## Sommaire

- `doc/fr/getting-started.md` : installation, prerequis et lancement local.
- `doc/fr/architecture.md` : architecture serveur/client/WASM.
- `doc/fr/python-api.md` : API publique de la bibliotheque `pywasm_ui`.
- `doc/fr/websocket-protocol.md` : protocole JSON WebSocket.
- `doc/fr/widgets-and-styling.md` : widgets, style et patchs UI.
- `doc/fr/testing-and-dev.md` : tests, build et workflow developpement.
